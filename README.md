# HSRP Booking Transport Site

This is a static website for HSRP (High Security Number Plate) booking, ready for deployment on Vercel.

## Features
- Modern, responsive design using Bootstrap 5.
- Dynamic payment selection based on vehicle type.
- Clean URL support via `vercel.json`.
- Optimized for fast loading using CDNs.

## Deployment
1. Push this folder to a GitHub repository.
2. Connect the repository to Vercel.
3. Vercel will automatically detect the static files and deploy.

## Structure
- `index.html`: Homepage
- `enquiry.html`: Application Form
- `payment.html`: Dynamic QR Payment Page
- `thank-you.html`: Confirmation Page
- `/images`: Asset folder
